# Experimental Skills

This folder contains experimental skills that are in development or testing.

Experimental skills may be incomplete, untested, or subject to change.

## Installing Experimental Skills

Use the skill-installer with the experimental path:

```
$skill-installer install the create-plan skill from the .experimental folder
```

Or provide the GitHub directory URL:

```
$skill-installer install https://github.com/openai/skills/tree/main/skills/.experimental/<skill-name>
```
