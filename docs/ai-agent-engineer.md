# AI Agent Engineer Domain

## Overview
Domain focused on AI agent orchestration, skill configuration, agent workflow optimization, and improving the AI agent development lifecycle.

## Responsibilities
- Skill orchestration and configuration
- Agent workflow optimization
- Agent tooling and tooling integration
- Testing and validation of agent behaviors

## Key Files
- `.opencode.json` - Agent configuration for orchestration
- `scripts/evaluate_skill.py` - Skill evaluation runner

## Agent Configuration
The `.opencode.json` provides:
- `ai-agent-engineer` primary agent - Main specialist agent
- `skill-evaluator` subagent - For evaluating skill quality

## Common Tasks
1. Add new agent configurations
2. Improve skill evaluation framework
3. Optimize agent workflows
4. Add test coverage for skills

## Notes
- Always use small, atomic PRs
- Label PRs with `ai-agent-engineer`
- Ensure lint/build passes before submitting
